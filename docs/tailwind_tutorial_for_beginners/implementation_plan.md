# Tailwind初心者向けチュートリアル 実装計画

## 目的
エンジニアやデザイナー以外の初学者（中学生でもわかるレベル）を対象に、Tailwind CSSの使い方をゼロから一歩ずつ学べるチュートリアルページをアプリ内に追加する。新しいタブ「Tailwind Tutorial」を設け、専門用語には必ず補足説明を行う。

## 追加するUI構成
1. **タブの追加**:
   - `App.vue` のナビゲーションに3つ目のタブ「**Tailwind Tutorial (初心者向け)**」を追加する。
   - `activeTab` の状態に `'tutorial'` を追加。
2. **チュートリアルコンテンツ (ステップバイステップ形式)**:
   - **Step 1: HTML（骨組み）とCSS（服）って何？**
     - Webページを「家」や「人」に例えた説明。Tailwindは「あらかじめ用意された服のタグ」という説明。
   - **Step 2: 色と大きさを変えてみよう（文字の装飾）**
     - `text-red-500`（文字を赤色にする）や `text-2xl`（文字を大きくする）の具体例とプレビュー。
   - **Step 3: 余白（よはく）の魔法（PaddingとMargin）**
     - `p-4`（内側のクッション）と `m-4`（外側のバリア）の違いを、箱（Box）の図解で解説。
   - **Step 4: 角を丸くして、影をつける（モダンなデザインへ）**
     - `rounded-lg`（角丸）と `shadow-md`（影）の効果。
   - **実践**: これらを組み合わせて「ボタン」を作ってみるデモ。

## ユーザーレビューが必要な項目
> [!IMPORTANT]
> チュートリアルの内容と構成について、以下の点を確認させてください。
> 1. **章立てのボリューム**: まずは上記の「Step 1〜4 ＋ 実践」という構成でよろしいでしょうか？（後からStepを追加することが可能な構造にします）
> 2. **デザインのトーン**: アプリ全体の「Soft Nordic Dark」を維持しつつ、学習ページとして読みやすいように明るいカード（図解エリア）やアニメーションを配置する方針でよいでしょうか？

## 検証・開発手順
1. `App.vue` に新しいデータ構造 `tutorialSteps` を定義する。
2. チュートリアル用のUIを描画するコンポーネント部分（`v-if="activeTab === 'tutorial'"`）をコーディングする。
3. 文言が中学生にもわかる平易な表現になっているか、専門用語（HTML, CSS, マージン, パディング等）に補足があるか確認する。
